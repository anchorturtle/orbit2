# 🪐 Orbit Planetary Player

> Cosmic High-Fidelity music player UI — dark-mode, glassmorphism-styled, built with plain HTML + Tailwind CSS CDN. No build step required.

## 🗂 Project Structure

```
orbit2/
├── index.html                  # Main app (music player UI)
├── netlify.toml                # Netlify deployment config
├── wrangler.jsonc              # Cloudflare Workers config
├── .gitignore
├── README.md
├── gallery/                    # Gallery assets folder
└── [media assets]              # MP3s, PNGs, JPGs, GIFs (see below)
```

## 🚀 Deployment

Deploys via **Netlify** (`netlify.toml`) and supports **Cloudflare Workers** (`wrangler.jsonc`).

## 🛠 Stack

- **HTML** — single-file app (`index.html`)
- **Tailwind CSS** — via CDN (no build step)
- **Glassmorphism / dark-mode** UI theme
- Hosted on Netlify / Cloudflare
